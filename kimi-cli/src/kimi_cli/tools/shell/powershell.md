Execute Windows shell commands. Each call runs in a fresh session; changes to environment or working directory do not persist. Chain commands with `;` and use `if ($?)` / `if (-not $?)` for conditional logic. stdout and stderr are combined. For long-running tasks, use `run_in_background=true` and manage with `TaskOutput`. Do not run interactive, blocking, or admin-elevated commands unless authorized. Available utilities include `cd`, `dir`, `copy`, `del`, `mkdir`, `findstr`, `where`, `tar`, `python`, `powershell`, and any binary on the system PATH.
